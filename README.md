#Launch
To launch and run HellaFresh.:
* Clone the repo onto your local machine
* ``` $ cd react-portfolio-project ```
* ``` $ cd backend ```
* ``` $ bundle install ```
* ``` $ rake reset ```
* ``` $ rake start ```
* Be sure to be running PostGres SQL application on local machine. (https://www.postgresql.org/download/)
* Enjoy!
